# Résumé 圖片資料夾

把履歷（Résumé / CV）的頁面圖片放在這個資料夾。

## 命名格式

使用小寫、兩位數編號，格式為：

```
resume-01.jpg
resume-02.jpg
```

- 編號要補零到兩位數（`01`、`02`…）。
- 建議尺寸比例為 **3:4**（直向 / A4 頁面），例如 1240 × 1754。
- 支援 `.jpg`、`.jpeg`、`.png`、`.webp`。若不是 `.jpg`，請一併修改
  `components/portfolio.tsx` 裡的副檔名。
- 網站會依照編號自動載入。找不到對應檔案時，會顯示佔位文字。

## 要增加或減少履歷頁數

在 `components/portfolio.tsx` 修改 `RESUME` 的數量（目前為 2）。
